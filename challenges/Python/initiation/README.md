# Average Of Numbers

## Overview

This small Python exercise asks the user for a number of grades, stores them in a list, and computes the average.

The goal of this script is not only to solve a basic problem, but also to show that I practice core Python syntax through simple and readable programs.

## File

- `averageOfNumbers.py`

## What The Program Does

1. asks the user how many grades they have
2. collects each grade with user input
3. stores the values in a Python list
4. calculates the average
5. prints the result

## Python Syntax Practiced

This exercise highlights several Python fundamentals that I use comfortably:

- function definition with `def main():`
- program entry point with `if __name__ == '__main__':`
- variable assignment
- integer conversion with `int()`
- list creation with `[]`
- loop with `for i in range(n):`
- list method usage with `append()`
- built-in functions such as `sum()` and `len()`
- arithmetic operations for average calculation
- console output with `print()`

## Why This Exercise Matters

Even if the problem is simple, it demonstrates important programming habits:

- breaking logic into steps
- collecting dynamic user input
- storing data before processing it
- using built-in Python tools instead of unnecessary complexity
- organizing code inside a dedicated `main()` function

## Run The Script

```bash
python averageOfNumbers.py
```
